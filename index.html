Work Orders: Users can create, manage, and complete various work orders.

Inspection Schedules: Includes comprehensive inspection scheduling and tracking. Directory Search: Provides a directory search feature for easy access to information on all properties the user works in.

Emergency Module: Equipped with an emergency module for immediate response and notifications for action.

Directory Search: Search feature for easy access to information on all properties the user works in.

Timeplay App

Over the years, TimePlay has become an immensely popular app in Toronto and Vancouver. I was part of the team working on this app from 2013 to 2018. Github repo

Networking Package

This repository offers a robust Swift package designed to simplify backend API interactions for your applications. The package works out of the box, providing developers with the tools to easily make API calls, handle authorization, and manage access tokens. It seamlessly refreshes expired tokens and continues API operations without interruption, allowing developers to focus on building features without worrying about API management. The package leverages the latest async/await APIs from Apple Swift, ensuring modern and efficient asynchronous programming.
