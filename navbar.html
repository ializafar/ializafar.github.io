Navigation Bar

<nav class="navbar slider half blur">

For mobile navigation use one of the following options
slider | overlay
Slider will slide the menu where as overlay will fade in the menu
half will keep the mobil menu to left half of the screen this is ideal for slider.

Use the sticky class on body to make the navigation stick to top of the screen

blur-background will make the mobile menue blur background

Change following variables to modify navbar theme

--navbar-background-color: #000;
--navbar-background-color-transparency: #000000dd;
--navbar-contrast-gradient-color: #ff0000;
--navbar-links-color: #fff;
--navbar-links-active-hover-color: #ccc;
